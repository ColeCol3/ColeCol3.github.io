Snake The Game

This is my Scratch project that is a take on the game 'slither.io'. The main goal of the game is to control the snake and navigate through the environment without hitting the edges. You collect the apples to increase your score.

To use it, Use the WASD keys to change the direction of the Snake. The game automatically creates clones or segments to extend the snake, which makes navigation more challenging over time. The game tracks the player's score and updates the score everytime an apple is touched

The most interesting part of the code includes The create clone of myself block. It generates new segments to simulate the growing snake..
